# Logo
- assets 폴더에 images 폴더 생성 후
- 로고로 지정할 이미지를 저장
- `_config.yaml` 파일에 수정
```
logo: # path of logo image to display in the masthead, e.g. "/assets/images/88x88.png"
```

# Breadcrumb Nevigation
- 네비게이션 메뉴 추가
- 주석 해제 후 true로 바꾸기

# Show Date
- 맨 아래의 defaults 에서
	- values에 `show_date`를 작성 후 true로 지정
		- `show_date : ture`
- `date_format`
	- `date_format: "%Y-%m-%d"` 맨 밑에 들여쓰기 없이 입력
	- 다른 옵션도 가능
