# this()
---
- 생성자 오버로딩이 많아질 경우 중복된 코드가 발생
- 생성자에서 다른 생성자 호출할 때 `this()` 사용
- 생성자의 첫줄에서 사용
```java
Car(){ }

Car(String model){
	this(model, "은색", 250);
}

Car(String model, String color){
	this(model, color, 250);
}

Car(String model, String color, int maxSpeed){  //this()에 의해 호출됨
	// 공통 실행 코드,
	this.model = model;
	this.color = color;
	this.maxSpeed = maxSpeed;
}
```
